# ⏰ Supported Timeframes

### Second-based Timeframes

| Value   | Display Name       | Description |
| ------- | ------------------ | ----------- |
| `'1S'`  | 1 Second / 1 秒    | 1 second    |
| `'5S'`  | 5 Seconds / 5 秒   | 5 seconds   |
| `'15S'` | 15 Seconds / 15 秒 | 15 seconds  |
| `'30S'` | 30 Seconds / 30 秒 | 30 seconds  |

### Minute-based Timeframes

| Value   | Display Name       | Description |
| ------- | ------------------ | ----------- |
| `'1M'`  | 1 Minute / 1 分    | 1 minute    |
| `'3M'`  | 3 Minutes / 3 分   | 3 minutes   |
| `'5M'`  | 5 Minutes / 5 分   | 5 minutes   |
| `'15M'` | 15 Minutes / 15 分 | 15 minutes  |
| `'30M'` | 30 Minutes / 30 分 | 30 minutes  |
| `'45M'` | 45 Minutes / 45 分 | 45 minutes  |

### Hour-based Timeframes

| Value   | Display Name       | Description |
| ------- | ------------------ | ----------- |
| `'1H'`  | 1 Hour / 1 小时    | 1 hour      |
| `'2H'`  | 2 Hours / 2 小时   | 2 hours     |
| `'3H'`  | 3 Hours / 3 小时   | 3 hours     |
| `'4H'`  | 4 Hours / 4 小时   | 4 hours     |
| `'6H'`  | 6 Hours / 6 小时   | 6 hours     |
| `'8H'`  | 8 Hours / 8 小时   | 8 hours     |
| `'12H'` | 12 Hours / 12 小时 | 12 hours    |

### Day-based Timeframes

| Value  | Display Name  | Description |
| ------ | ------------- | ----------- |
| `'1D'` | 1 Day / 1 日  | 1 day       |
| `'3D'` | 3 Days / 3 日 | 3 days      |

### Week-based Timeframes

| Value  | Display Name   | Description |
| ------ | -------------- | ----------- |
| `'1W'` | 1 Week / 1 周  | 1 week      |
| `'2W'` | 2 Weeks / 2 周 | 2 weeks     |

### Month-based Timeframes

| Value    | Display Name    | Description |
| -------- | --------------- | ----------- |
| `'1MON'` | 1 Month / 1 月  | 1 month     |
| `'3MON'` | 3 Months / 3 月 | 3 months    |
| `'6MON'` | 6 Months / 6 月 | 6 months    |
